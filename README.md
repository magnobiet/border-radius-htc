# Border Radius HTC v3.0.1

Border Radius for Internet Explorer 6, 7 & 8 support

## Usage

    .rounded {
    	-khtml-border-radius: 5px;
    	-webkit-border-radius: 5px;
    	-moz-border-radius: 5px;
    	border-radius: 5px;
    	behavior: url(border-radius.htc);
    }
